⚙️ Installation

Clone the repository:

git clone https://github.com/omidhabib/portfolio.git
cd portfolio

Install dependencies:

pip install -r requirements.txt

Run the development server:

python manage.py runserver

Open in browser:

http://127.0.0.1:8000/
